Panduan

1. Pilih metode analisis data

Sebelum melakukan proses analisis data, pilih metode yang ingin dilakukan terlebih dahulu, dengan menekan opsi “Pilih metode analisis data”.

Setelah itu, akan muncul pop-up berupa pilihan analisis data. Silahkan pilih metode analisis data yang ingin kamu lakukan. Seperti contoh di bawah, “Uji Homogenitas” telah dipilih.

2. Menginput data

Saat menggunakan SPSS, tampilan data terlihat seperti ini:

Dalam aplikasi Laboratorium Statistika, kamu perlu memasukkan data seperti ini:

Angka yang dimasukkan berupa bilangan numerik yang di pisahkan dengan spasi (gunakan titik untuk pemisah desimal, jangan koma).

Saat kamu membuka fitur “Analisis Data”, secara otomatis akan muncul variable/grup pertama yaitu “group_1” yang sudah siap untuk dimasukkan data. “group_1” merupakan nama dari variabel.

Kamu dapat mengubah nama variabel dan memasukkan data ke dalamnya.

Kamu juga dapat menambahkan variable data baru dengan menekan tombol “Tambah Data”.

Selanjutnya, kamu dapat menambahkan nama untuk variable baru dan memasukkan data ke dalamnya.

3. Menjalankan analisis data

Setelah memilih metode analisis yang sesuai dan sudah memasukkan data, selanjutnya kamu dapat memilih data yang akan di proses dengan menekan kotak di sebelah kiri masing-masing variabel hingga muncul centang yang menandakan data telah terpilih.

Sebelum menjalankan proses analisis, harap periksa kembali data yang telah dimasukkan dan ketentuan data dari masing-masing metode yang tampil pada layer di bawah tombol “Tambah Data”. Selain itu, perhatikan metode yang digunakan apakah memerlukan tambahan nilai seperti nilai signifikansi/alpha (Secara default, nilai signifikansi = 0.05).

Jika sudah sesuai, maka analisis data dapat dilakukan dengan menekan tombol “Run”. Hasil analisis data akan muncul pada kotak “Hasil”. Jika ingin melakukan analisis dengan data atau metode yang berbeda, kamu dapat mengulangi langkah-langkah di atas. Semua hasil analisis akan di jalankan di dalam kotak “Hasil”.

4. Menyimpan/menghapus hasil analisis

Kamu dapat menghapus hasil analisis dengan menekan tombol “Hapus”, maka semua hasil analisis yang telah dilakukan akan terhapus dan akan menampilkan status seperti di bawah ini.

Apabila kamu ingin menyimpan hasil analisis, kamu dapat menekan tombol “Simpan”. Kamu dapat memilih lokasi penyimpanan dan memberikan nama file yang kamu inginkan. Jika sudah selesai, klik “Save”. File hasil analisis yang tersimpan berupa format “.txt”.
